Phase 1

Initial Startup & EXOR Remotes

Pre-requisite Checks

Before beginning, ensure you grab the electrical fire extinguisher from Glen's office. Double-check all bus bar phasing and wire phasing in each drive. Ensure wires are securely on prepare drive and ethernet cables are seated properly.

1

Base Unit / X-Base Configuration

Open Microsoft Edge and navigate to HTTPS://192.168.0.1/machine_config. (If the page doesn’t open, click 'Advanced' and 'Continue to site'). Make sure your local computer IP is in the 192.168.0.# range.

Login & Localization
  • Default username is admin. Default password is admin or Admin123$.
  • Navigate to the Authentication tab, click edit, and change the admin password to Tarus123!. Then change the User password to Tarus123!.
  • Go to the Localization tab and set Country Code to United States of America.
Network Setup
  • Navigate to the Network tab > Wireless networking and press edit. Verify it is Enabled. Set Safety WiFi network to ON.
  • Go to the Network Interfaces tab and configure the following:
    • Eth 0 → 192.168.214.11
    • Eth 1 → 192.168.214.12
  • Hit save. You will hear it beep because the system is rebooting. (Once complete, you will need to change your laptop's IP address to match the new subnet and log back in using the new IP URL).
2

Base Unit OS Management & Routing

Bridge, Router & Port Forwarding
  1. Go to the Service tab on the left. Click Bridge switch, hit Edit, and Enable it. The system will reboot (make sure device IP is 192.168.214.11).
  2. Navigate to Router / NAT / Port Forwarding. Enable 1:1 Nat rules with the following settings:

X5

  • Hit save and done. Under Services, go to VNC Server and enable Auto Start to YES.
Firmware Updates (Management)
  • Go to Management and click on Main OS. Browse for the image file ending in .gz. Browse for the .md5 file. Click Update.
  • Repeat this exact process for the Config OS. Note: Every time you do these updates, the firmware system will reboot.
3

Handheld Remote Setup

Initial Boot & Passwords
  1. Hold F2 and F5 to turn on, and turn the hand wheel to get to the boot screen.
  2. Go to System Settings. Change the password to Tarus123!. Do the same for the Admin account. The system will reboot.
  3. Turn the wheel to get back to the boot screen (without holding F2/F5 this time). Log into Admin with user admin and pass Tarus123!.
  4. Change the Country Code to USA.
Network & Services
  • Go to Network Setting > Wireless networking. Hit Edit, enable Wireless Networking, and enable Safety WiFi network. Hit the checkmark to save.
  • Go to Services. Enable the SSH Server and enable Auto Start. In the same tab, enable the VNC Service and enable Auto Start. Exit Service Settings.
Startup Sequence & Firmware
  • Enter Startup Sequence. Select Install → MMT → USB Memory. Look for 'Update Package', move it to the right, and hit done.
  • Go to Management > Main OS. Verify firmware version 1.3.937. If not, click update, browse for the .gz image file, and update. Do the same for Config OS using the config file.
Display & Ping Setup
  • Go to Service Settings > Display. Turn brightness on high max, set a 10-minute shut off timer, and perform a touch calibration.
  • When you boot up the HMI screen, enter the password. A settings screen will prompt you to ping your remote. Ensure both checkboxes are set.
  • Type IP Address 172.27.72.1 and ping. It will prompt that the remote couldn't connect. Turn off the remote at this point because the PC needs the project loaded.
4

J-Mobile Project Deployment & Pairing

  1. Go to the Run tab in J-Mobile on your laptop.
  2. Click Manage the Target and select Install Run Time. Download runtime to the base unit.
  3. Once complete, do a full reset of the machine.
  4. After the machine is rebooted, download the project. (Don't panic if the download freezes momentarily).
  5. Turn the remote back on. You will have to input the UID for the base (this can be found in the system setting tab on the base unit).
5

Remote Desktop Access

For the PC located in the tower:

  1. Set up password as Tarus123.
  2. Search for and enable Remote Desktop Settings.
  3. From the Desk PC, open Remote Desktop Connection. Type in the target computer's name (Find in "This PC").
  4. Login User is Tarus. Password is Tarus123 (or the one you just set up).
Phase 2

Operate & Network Setup

Important Key Requirement

If asked for a key Switch 0 when loading an archive, input the password SUNRISE.

1

Operate Configuration & OPC UA

Once you have loaded an archive into Operate, proceed with the following OPC UA setup:

  1. Open Operate and add the OPC UA License: Access mymachine opc ua.
  2. Go to Setup (Page 2).
  3. Navigate to Network > OPC UA > Settings > Change.
  4. Set Company Network: Check.
  5. Set System Network: Uncheck.
  6. Set Additional Network #1 address to: 192.168.214.200 (Matches PC IP address).
  7. Set TCP Port Number: 4841.
  8. Set Message Encryption: Check all boxes (None, 128bit, and 256bit).
  9. Check Activate OPC UA.
  10. Reboot the machine.
OPC UA Connection Notes
  • OPC will only be on and connect if Operate is currently open.
  • Ensure the system time is correct, or it may fail to connect.
  • Fallback Port Change: If you need to change a port, go to the J-Mobile app on your laptop → Protocols → Look for OPC UA Client → Click the box under Configurations → Change the port number on the prompt → Press OK → Download again.
2

Windows Firewall Setup

After rebooting and activating OPC UA, configure the Windows Firewall to ensure proper connectivity.

  • Initial Prompt: If you receive a Windows Firewall popup/alert upon reboot, ensure you enable BOTH Private and Public networks.
  • Open Windows Settings and search for Windows Defender Firewall.
  • On the left pane, click Advanced settings.
  • Under "Public Profile is Active", click on Windows Defender Firewall Properties.
  • Under the "Firewall State" section, find Protected Network Connections and press Customize.
  • Uncheck the Ethernet option corresponding to the machine connections.
3

Machine Setup & MCP Switches

  1. Change the name of the I/O (must be physically plugged into I/O).
  2. STI Mod Estop (Red one): Change dip switches. Push both in G2 -> G1 and M -> A (Note: Not applicable on portables).
  3. MCP Switches: Ensure switches 7 and 8 are ON. All rest must be OFF.
  4. Change the name of the machine within the interface.
  5. Check and load in all necessary licenses. Disable the external HMI.
4

Connecting the Remote

Remember to keep switch 8 ON until connected.

  • Make sure the big white switch is down.
  • Check that the I/O unit is properly reacting to the prepared inputs from the remote.
  • Ensure E-stops function correctly and can be cleared using the blue button.
Phase 3

Axis Calibration

X

X Motor Setup

Make sure X1 and X2 are active. Verify X is counting properly before proceeding.

Start Motor Feedback
  • Set relative 0 using parameter 34210.
  • Try to get full X positive limit.
X on Scales
  1. Load in a K value in 34300 (1) (e.g., 192, 332, 368, 384).
  2. Go to encoder 2 and put in the K value (offset).
  3. Go into reference mode and reference it. If there is a negative change, alter p36100 to -10,000. Repeat this 2-3 times to ensure it doesn't jump around. (If encoder is jumping the wrong way, change 34320 (1)).
  4. Set 34090 (1) to 0.
  5. Go roughly to soft limit 0 and execute Reference +. Change p36100 back to -0.1.
  6. Grab the number that populates when referencing from the desired zero position. Put it into 34090 and re-reference.
  7. Jog back to 0 slowly. Do calculations on what the number is after reference: Measure distance between machine location and true zero. +/- that number from the value in 34090 to establish true zero.

Z

Z Motor Setup

  • Activate z motor via p105. Make sure Z is counting the right direction.
  • Set 0 with 6mm from hard stop. Match Z travel to desired travel.
Z Scales
  • Change measuring system to sys 2.
  • Set limit to 0 and verify z travel.
AC

A/C Motors Setup

General limits: +/- 105.1 for A, +/- 180.1 for C. Move the A/C head by hand to confirm the correct axis is counting and in the right direction.

Portables Exception: For C rotation, look at the PLC I/O tags and put that number in parameter 14510 (6).

Motor Feedback
  • Activate motor p105. Hand move to roughly 0 degrees.
  • Turn drives ON and commutate angle p1990. Get a number close to zero.
  • Move A/C to soft limit. If A is moving the wrong way, alter p32100.
A/C Scales
  • Before changing to measuring sys 2, put A/C to zero.
  • Change to measuring sys 2, perform a Reference, and verify all limits.
S

Spindle

  • Activate motor p105.
  • Commutate angle p1990. (Note: Portables do NOT commutate the spindles).
  • Run at 10rpm. Ensure proper count direction (Clockwise from behind, counterclockwise in front).

Phase 5

Lift & Fooke Tables

1

Lift Tables Base Configuration

  • Go online at the table and ensure the network address matches Gen 3.
  • Download Cu_5.
  • Lift 1: In Parameter view, verify P10 is "0 ready". In Function view > Technology function > Basic positioner > Referencing, set Absolute encoder to Set.
  • Lift 2: Ensure R21905 is 0. For R21902, cycle from 0 → 1 → 0.
  • Rotary: In Parameter view, verify P10 is "0 ready". In Function view > Referencing, hit Set. Save via blue floppy disk icon, go offline, upload, go back online, and release E-stop to test.
Troubleshooting Tip

If having problems with Lift Tables, look at CU_5 → Parameters → Function view → Input → Verify DI 16 and DI 17.

2

Lift Tables (Motors Engaged)

  1. Retest motors and logic. (If running away, check wires). Deactivate rotary (hit E-stop, check p105).
  2. Ensure slave is roughly 2 inches higher than master. Have Mark/Dale connect the motors.
  3. Reference both motors (hit E-stop), then "Hit the sticks" to drive both motors up into slots.
  4. Once slave is touching, deactivate it (p105) and drive master into the slot. Reactivate slave (E-stop).
  5. Set Limit to 850,000.
    • Lift 1: Reference Lu
    • Lift 2: Set limit 21905. Change 21902 0→1→0.
    • Set a fake limit in rotary.
  6. Take pins out, bring table down to fit into the slot, and verify. E-stop machine and reset zero for Lift 1 and Lift 2.
  7. Take fully up to limit, bring down to fit in pocket, reactivate rotary, set rotary 0, go fully up and test.
3

Fooke Table

* Same general procedure as a normal lift table, but there is no rotary component. *

  • Retest motors/logic. Ensure slave is 2 inches higher than master before connecting.
  • Reference both motors, drive up into slots. Deactivate slave when touching, drive master in, reactivate slave.
  • Set Limit to 794,000. Take pins out, bring table down to slot, have Mark check.
  • E-stop machine, reset zero for both lifts. Take fully up, bring back down to pocket, and test.
Phase 6

Testing, Legs & Portables

1

Weight Testing (r33)

If cutting quality is bad, look at parameter (r81).

Without Weight
  • Motor 1: Up 9.7% | Down 26.0%
  • Motor 2: Up 9.3% | Down 25.5%
  • Rotary CCW: 4.3% | CW: 4.2%
With Weight
  • Motor 1: Up 25.7% | Down 22.2%
  • Motor 2: Up 23.0% | Down 21.7%
  • Rotary CCW: 6.2% | CW: 5.7%

Go to assembly's computer to put in the final measurements.

2

Leveling Legs Limit

  • Go online (match IP address, either CU320 or old FW). Open parameterization.
  • Open Rotate & Optimize for Leg 1. Slowly raise it in at 10 rpm while watching torque.
  • Back off 6000lu. Put -6000lu in at reference @ hard stop.
  • Set limits to 90170 lu or 90.17mm.
  • In Parameter view, change P971 to 1. If it doesn't revert to 0, manually change it back.
  • Save via blue floppy, go offline, upload, cycle power off/on. Check limits with Tarus NC on their side via Auto Leave.
Leveling Leg Notes

200040 = Level legs over torque: Usually just manually lower the closest leg next to it to take weight off the one leg.

200050 = Level leg limit.

Portables Overview

Note: Do Gen 3 set up without the EXOR remotes.

Ohm out each motor exactly like Gen 3.

Test Legs
  • Plug into the CU32 (127 port) and go online.
  • Open each leg and go to Rotate and Optimize.
  • Parameter 1990: Change 0 → 1 to take the commutation angle.
  • Parameter 1520 (High Torque): Change 3.32 → 1.32. Once hit, revert to 3.32.
  • Parameter 1521 (Low Torque): Change -3.32 → -1.32. Once hit, revert to -3.32.
  • Under Rotate and Optimize jog back until torque spike, set limit there.
  • Use parameter 971 to save drive parameters 0-1 (Save RAM to ROM).
