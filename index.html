5-Axis Commissioning

Phase 1

Initial Startup & EXOR Remotes

Pre-Startup Checks

Before beginning, grab the electrical fire extinguisher from Glen's office. Double-check all bus bar phasing and wire phasing in each drive. Ensure wires are securely on prepare drive.

  1. STI Module Estop (Red TapeSwitch Device): Change dip switches. G2 → G1 and M → A. Note: Not applicable on portables.
  2. MCP Switches: Ensure switches 7 and 8 are ON. All rest must be OFF.
  3. NCU Rotary Switch: Ensure the Rotary Switch is set to 0 on front of NCU.
  4. NCU Ethernet Cables: MCP → X120, I/O → X150 (P1 or P2).
  5. Remove Magnets: From Y Pos and Z Pos Schneeberger encoders.
1

Base Unit / X-Base Configuration

Open Microsoft Edge and navigate to HTTPS://192.168.0.1/machine_config. Note: If the page doesn’t open, click 'Advanced' and 'Continue to site'. Make sure your local computer IP is in the 192.168.0.# range.

Login & Localization
  1. Default username is admin. Default password is admin or Admin123$.
  2. Navigate to the Authentication tab, click edit, and change the admin password to Tarus123!. Then change the User password to Tarus123!.
  3. Go to the Localization tab and set Country Code to United States of America.

    • Eth 0 → 192.168.214.11
    • Eth 1 → 192.168.214.12
  3. Hit save. You will hear it beep because the system is rebooting. Note: Once complete, you will need to change your laptop's IP address to match the new subnet and log back in using the new IP URL.

Phase 3

Axis Calibration

X

X Motor Setup

Motor Feedback
  1. Ensure X1 and X2 are Active via drive parameter p105.
  2. Set Limits on Motor Feedback: Check PLC FC_Axis 1 for MeasSys1 = ON.
Scale Feedback
  1. Load K value into 34300[1] (e.g., 192, 332, 368, 384).
  2. Go to Encoder 2 Settings and add K value.
  3. Set MD34090 = 0.
  4. Activate RefPoint Mode and reference X by jogging. Repeat referencing to ensure numbers are counting correctly.
  5. Adjust 34320 from -1 to 1, or invert encoder in drive settings.Note: This changes counting direction.
Set Home Offset
  1. Move to desired negative limit. Set MD34090 = value of X position.
  2. Verify Negative and Positive Limits.
Y

Y Motor Setup

Y Motor Feedback
  1. Ensure the hydraulic pump is ON. Verify Pressure/Flow set points are correct on Hydraulic IFM switch.
  2. Put starter at 1.6 and flip breaker.
  3. Put Y Carrier on crane and Activate Y Drive via p105. Watch hydraulic tank pressure to ensure pumping stops at 1180psi, and cycle pump a couple times.
Set Negative Limit
  1. Take down to 4mm from hardstop touching. Note: This creates a physical buffer from a hard stop.
  2. Set 34210 to 1.
  3. Switch to RefPoint Mode and press Y+ Jog. 34210 will switch to 2 and the axis will be referenced.
  4. Ensure positive travel limit p36110 is set properly (e.g. 2400.1 and -0.1).
Y on Scales
  1. Add Measuring System 2: Setup > Drives system > Drives > Change. Hit next step until you reach Encoder settings, add Encoder 2 and finish steps.
  2. Assign Meas System 2 to Y Axis: Go to Axis assignment > Axis 2/Y. Select Encoder 2 and Measuring system 2.
  3. Set Limits on Scales: Set 34210 = 1, put machine into RefPoint Mode and jog positive.
Z

Z Motor Setup

  1. Activate Z1 and Z2 motors via p105. Make sure Z is counting in the right direction. If axis is moving the wrong way, alter p32100.
  2. Move to 0-limit with 6mm from hard stop and Reference. Set 34210 = 1, put machine into RefPoint Mode and jog positive.
Z Scales
  1. Change measuring system to sys 2.
  2. Set limit to 0 and verify z travel.
AC

A/C Motors Setup

Standard limits: +/- 105.1 for A, +/- 180.1 for C. Move the A/C head by hand to confirm the correct axis is counting and in the right direction.

Portables Exception: For C rotation, look at the PLC I/O tags and put that number in parameter 14510 (6).
Motor Feedback
  1. Activate motor via drive parameter p105. Move by hand to roughly 0 degrees.
  2. Turn drives ON and commutate motor angle via p1990 = 1. Do this 3-4 times to get the pole angle difference number in p1984 close to zero. Note: If the number keeps bouncing around, you may have an encoder or phasing problem.
  3. Test A/C soft limits. If axis is moving the wrong way, alter p32100.
A/C Scales
  1. Before changing to measuring sys 2, put A/C to zero.
  2. Change to measuring sys 2, perform a Reference, then verify movement as well as limits.
S

Spindle

  1. Activate motor via drive parameter p105.
  2. Turn drives ON and commutate motor angle via p1990 = 1. Do this 3-4 times to get the pole angle difference number in p1984 close to zero. Note: If the number keeps bouncing around, you may have an encoder or phasing problem.
  3. Crucial: The spindle will need to be commanded to move, either via Jog +/- or by setting a desired speed, before the drive will enable for commutation. Note: Portables do NOT commutate the spindles.
  4. Run at 10rpm. Ensure proper count direction (Clockwise from behind, counterclockwise in front).
  5. Find tool/tool holder to put in spindle and run up to 10,000 rpm to test full speed.

Auto-Servo Tuning

Navigate to the Optimization screen, select an axis for tuning and click Auto-Servo Tuning.

Phase 5

Lift Tables & Weight Test

1

Lift Tables Base Configuration

Go Online & Network Setup
  1. Choose network address. Set Type of PNG/PC interface = PE/IE.
  2. Set PG/PC interface = Realtek PCIe Gbe family controller.
  3. Set First gateway = show everything.
  4. Select Targeted device = show all.
  5. Ensure IP address for your laptop is 169.254.11.# (the last octet must be higher than 10).
  6. Download the project to CU_S.
Axis Initialization
  1. Lift 1: In Parameter view, verify P10 is 0 ready and set P105 to 1. In Function view → Technology function → Basic positioner → Referencing → Absolute encoder, hit Set.
  2. Lift 2: In Parameter view, verify P10 is 0 and set P105 to 1. To reference: ensure R21905 is 0, then change R21902 from 0 → 1 → 0.
  3. Rotary: In Parameter view, verify P10 is 0 ready and set P105 to 1. In Function view → Technology function → Basic positioner → Referencing → Absolute encoder, hit Set.
Save & Test
  1. Save via the blue floppy disk icon.
  2. Go offline, Upload, Go back online, and Release E-stop to Test.
Troubleshooting Tip

Note: If having problems with Lift Tables, look at CU_S → Parameters → Function view → Input/output → Iso digital inputs. Verify DI 16 (make sure top box is green) and DI 17 (make sure top box is green).

2

Lift Tables (Motors Engaged)

  1. Retest motors and logic. Note: If running away, check wires.
  2. Deactivate rotary (Hit E-stop, set Parameter 105 = 0).
  3. Ensure slave is roughly 2 inches higher than master. Have Mark/Dale connect the motors.
  4. Reference both motors (make sure it is E-stopped).
  5. "Hit the sticks" to drive both motors up into slots. Note: The guys install rods into the spiral lift—whatever diameter fits the holes in the spirals—this prevents the spiral lift from spinning in place and allows it to actually drive upward.
  6. Once slave is touching, deactivate it (Hit E-stop, set P105 = 0).
  7. Drive master into the slot.
  8. Reactivate slave (Hit E-stop, set P105 = 1).
  9. Set limit to 850,000:
    • Hit E-stop.
    • Lift 1: Function view → Technology function → Basic positioner → Referencing → Absolute encoder → Set to 850,000 lu, then hit Set.
    • Lift 2: Parameter 21905 set to 850,000. Change 21902 from 0 → 1 → 0.
  10. Take pins out (Move up slowly until they come out).
  11. Have Dale level the table.
  12. Bring table down slowly until it fits into the slot.
  13. E-stop machine and Reset zero:
    • Lift 1: Function view → Technology function → Basic positioner → Referencing → Absolute encoder → Set to 0 lu, then hit Set.
    • Lift 2: Parameter 21905 set to 0. Change 21902 from 0 → 1 → 0.
  14. Take fully up to limit, make sure table is 37.500 from base then bring down to fit in pocket.
  15. Have Dale measure for shims.
  16. Reactivate rotary (Hit E-stop, set Parameter 105 = 1).
  17. Set rotary 0 (Hit E-stop. Function view → Technology function → Basic positioner → Referencing → Absolute encoder → Set 0).
  18. Go fully up.
  19. Test Rotary:
    • 90 degrees
    • 180 degrees (Bring down to make sure it fits in pockets, then take back up)
    • 270 degrees
    • 0 degrees
3

Weight Testing (r31)

Note: If cutting quality is bad, look at parameter (r81).

Without Weight
  • Motor 1: Up # | Down #
  • Motor 2: Up # | Down #
  • Rotary CCW: # | CW: #
With Weight
  • Motor 1: Up # | Down #
  • Motor 2: Up # | Down #
  • Rotary CCW: # | CW: #

Go to assembly's computer to put in the final measurements.

Crucial Action:

If the Weight Test yields percentages way outside of the given baselines (e.g., Up 9.7% / Down 26.0%), call and let the assembly manager know immediately.

Leveling Legs Limit

  1. Go online (match IP address, either CU320 or old FW). Open parameterization.
  2. Open Rotate & Optimize for Leg 1. Slowly raise it in at 10 rpm while watching torque.
  3. Back off 6000lu. Put -6000lu in at reference @ hard stop.

Portables Overview

Note: Do Gen 3 set up without the EXOR remotes.

Ohm out each motor exactly like Gen 3.

Test Legs
  1. Plug into the CU32 (127 port) and go online.
  2. Open each leg and go to Rotate and Optimize.
  3. Parameter 1990: Change 0 → 1 to take the commutation angle.
  4. Parameter 1520 (High Torque): Change 3.32 → 1.32. Once hit, revert to 3.32.
  5. Parameter 1521 (Low Torque): Change -3.32 → -1.32. Once hit, revert to -3.32.
  6. Under Rotate and Optimize jog back until torque spike, set limit there.
  7. Use parameter 971 to save drive parameters 0-1 (Save RAM to ROM).

Editor Tutorial & Manual

How to Edit the Master Guide

This guide is fully interactive. You do not need database access or special coding tools to update the shop floor procedures. You can edit it directly from your browser.

  1. Turn on Edit Mode: Click the Enable Edit Mode button in the bottom left corner of the sidebar.
  2. Modify an Existing Step: When Edit Mode is on, hover over any card to reveal the Edit tools. Click Edit to begin typing directly inside the card. Click Save when you are done.
  3. Manage List Items: When editing a card, click anywhere inside the text of a specific step. Use the floating formatting toolbar to + Step (add new line), Move ↑/↓, or Delete that specific line instantly!
  4. Log Your Activity: Navigate to the Activity Log tab, click "Admin Edit", and use the password to document what you changed! You can use the formatting tools to add bullets.
  5. Save Your Work: Once you are done making changes, click the Save Master Guide button in the sidebar. This will instantly download a brand new HTML file. Send the exported HTML to the Applications Dept.

Progress Tracking

You can now use this guide to track your exact progress during a startup!

  1. Click the checkbox on the left side of any step you complete to mark it off.
  2. A progress banner will appear at the bottom of your screen.
  3. If you made a mistake or want to wipe the slate clean, click the red Discard button.
  4. If you need to leave the job or close your laptop, click the large green Save Progress button.
  5. This will download a personal HTML file to your Downloads folder. When you open that file tomorrow, all your checkboxes and typed notes will still be there exactly how you left them!

How to Export a "Custom Field Guide"

If you have a tech out in the field who only needs to see 2 or 3 specific steps (e.g. how to reference the Y axis), you can create a Custom Field Guide just for them.

  1. Turn on Edit Mode.
  2. Check the Boxes: Look at the top right of every step card. Check the box that says ☑ Include in Field Guide for the specific steps you want to send.
  3. Add Custom Notes: Click "Edit" on the checked cards and type in any custom notes or warnings for the field tech.
  4. Export: Click the Export Custom Field Guide button in the sidebar.

This will instantly download a stripped-down HTML file that ONLY contains the cards you checked. It completely removes the editing tools and empty tabs so the field tech gets a clean, simple guide. Send them that file via Teams or email!
